iNotebook-
iNotebook is a full-stack notes management web application built using the MERN stack.
It allows users to securely create, edit, and manage personal notes with authentication and cloud-based storage.

Live Demo :

Frontend (Vercel):https://i-notebook-pi-five.vercel.app/
Backend (Render): https://inotebook-backend-ki7k.onrender.com

Tech Stack :

Frontend: React, React Router, Bootstrap (Vercel)
Backend: Node.js, Express, JWT, bcrypt (Render)
Database: MongoDB Atlas

Features :

User authentication (JWT)
Secure password hashing
Protected routes
Create, edit, and delete notes
User-specific data isolation
Fully deployed frontend and backend

Deployement overview :
React (Vercel) → Node/Express API (Render) → MongoDB Atlas

Running locally :
git clone https://github.com/wamanvinit3817/iNotebook.git
cd iNotebook

# Backend
cd Backend && npm install && npm start

# Frontend
cd inotebook && npm install && npm start
